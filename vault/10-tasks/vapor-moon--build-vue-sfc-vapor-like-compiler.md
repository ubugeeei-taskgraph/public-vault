# Build the vapor-moon Vue SFC Vapor-like compiler

## Outcome

Create a concrete compiler prototype that tests whether Vue SFC and Vapor-style ideas can be translated into a MoonBit-oriented implementation story.

## Notes

This is intentionally ambitious, so the value is not only in getting code to run but in exposing where the MoonBit model and Vapor-style compilation align or resist each other.
