# Implement graphql.mbt

## Outcome

Turn the MoonBit GraphQL exploration into a tangible package that can expose schema authoring, resolver structure, and execution tradeoffs through a real API.

## Notes

This is the implementation-facing step inside the broader GraphQL framework investigation. The goal is not only to make something runnable, but to learn which parts of GraphQL map naturally onto MoonBit and which parts create pressure on the language or package design.
