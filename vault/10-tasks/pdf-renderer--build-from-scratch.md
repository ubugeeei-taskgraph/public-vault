# Build a PDF renderer from scratch

## Outcome

Understand where PDF complexity really lives by working through parsing, representation, and drawing behavior without hiding behind existing renderers.

## Notes

This is likely to be one of the more demanding scratch implementations, so the value is as much architectural understanding as it is a finished renderer.
