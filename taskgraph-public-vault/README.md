# Public vault conventions

- every task is a Markdown file with task frontmatter
- task files live directly under `10-tasks`
- every public task uses `visibility: public`
- weekly and monthly attention is stored in the single task property `focus`
- notes under `20-focus` are optional curated views, not the source of truth
- `90-index/home.md` is the primary landing note for day-to-day use
- `90-index/public-scope-disclosure.md` explicitly explains that this vault is only a public subset
- `90-index` contains navigational notes and generated indexes

Sensitive execution detail belongs in `private-vault`, even when it supports a public initiative.
